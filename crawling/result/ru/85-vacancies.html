Описание вакансии

О компании и команде

Работа в компании IndigoLab, это свобода в реализации решений и индивидуальная ответственность за решаемую задачу. Проявляя свой талант, Вы сможете находить наилучшие решения, воплощать передовые практики и инновационые подходы. Нам интересно работать с людьми, которым важно перед исполнением задачи, ее осмыслить, понять и принять цели  для бизнеса, участливо предлагать свой вариант реализации.

Ожидания от кандидата

Команда IndigoLab -  люди с высоким интеллектуальным потенциалом. Способные быстро и качественно разбираться с новой информацией. Критически мыслить, работать в команде, уметь конструктивно развиваться, развивая компанию. Мы нацелены находить простые эффективные решения, вместе с тем стремимся осваивать и внедрять инновационные подходы. Имеем амбиции соревноваться с лучшими ит-командами России.

Условия работы

Indigolab, разработчик ит-решений для  работы с маркетплейсами. Резидент «Сколково», ведущий технологический партнер: Wildberries, Ozon,  ЯндексМаркет, AliExpress, СБерМегаМаркет.

Мы ожидаем, что Вы:

  • Понимаете как работает HTTP
  • Уже ранее работали с REST API
  • Имеете опыт работы с PHP 7+
  • Знаете фреймворк Symfony,
  • Умеете работать с базами данных (Mysql/Postgresql)
  • Знаете git, умеете работать в командной строке

Задачи, которые Вас ждут:

  • Работа с внешними API различных маркетплейсов
  • Обработка больших объемов данных
  • Автоматизация рутинных процессов
  • Разработка сервисов для работы с маркетплейсами
  • Интеграции с различными клиентскими сервисами

У нас вы сможете:

  • Выбирать любые инструменты
  • Самостоятельно принимать решения
  • Получить огромный опыт работы с REST API
  • Учиться у более опытных коллег
  • Забыть о бюрократии и просто писать код
